Introduction

Thank you for buying NextGenSprites and supporting me!

NextGenSprites is a Modular-Shader for Sprites that brings many new features like
Normal mapping (Curvature), Reflections, Dissolve and much more. By being a "Modular-Shader" 
all Standard Features can be toggled - only enable what you really need! Furthermore the 
NextGenSprites Package delivers some cool Effect Shaders for Glass, Lava and Liquid. And if you 
like to change values at runtime, just use the provided C# Helper Methods or Playmaker Actions!

-------------------------------------------------------------------------------------------------
Requirements

- Unity 2017.4 LTS
- Pixel Shader 3.0

-------------------------------------------------------------------------------------------------
Installation and Content

Just leave NextGenSprites in your Assets root folder. Changing location may break Inspector
Layouts. The Demo Scene and Extras Folder can be deleted safely.
In the Extras Folder you get:

*	Playmaker Actions: All the Actions you need to manipulate the Materials at runtime.

External Content [Free Download at NextGenSprites Wiki]

*	Bootstrap Assets: Contains Textures/Patterns and Normalmaps to get you kickstarted with NGS.

In the Utils Folder you find:

*	Custom Material Editors
*	Editor Helper Widget
*	NextGenSprites Shortcut Methods
